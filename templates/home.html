Movies Recommendation System

Web Interface Endpoints

  • Popularity-Based Recommendations
  • Content-Based Recommendations (Sample movie: "Interstellar")
  • Collaborative-Based Recommendations (Sample user: 99)

API Endpoints

  • /api/popularity - Get popularity-based recommendations
  • /api/content - Get content-based recommendations (Path params: movie_name="Interstellar", num_movies=10)
  • /api/collaborative - Get collaborative-based recommendations (Path params: user_id=99)

API Documentation

  • Swagger UI - Interactive API documentation
  • ReDoc - Alternative API documentation

Use the API endpoints to retrieve data in JSON format for integration with other applications.
